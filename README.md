## About the project

- This classifier was created using the Fastai library with a pretrained model - Resnet34 to predict if a patient has pneumonia or not based on chest X-rays. The model ulitilize the concept of transfer learning with Resnet34 architecture and was fined tune to accomodate this case. Transfer learning allows more time-saving and computational capacity. The final result was an image classification project that was able to detect pneumonia with ~93% accuracy overall

## Dataset
- You can download the [Kaggle Chest X-ray Images (Pneumonia)](https://www.kaggle.com/paultimothymooney/chest-xray-pneumonia/tasks)
